# Brute Force (완전탐색)

[메인으로 돌아가기](https://github.com/tony9402/baekjoon)

풀어보면 좋을 문제는 추천 문제에 체크(:heavy_check_mark:) 해놨습니다.

추천 문제 아닌 나머지는 나머지를 난이도 섞었습니다.

완전탐색 중 백트래킹 있었던 문제와 최대한 겹치지 않도록 했습니다.

다른 알고리즘 풀었어도 완전탐색으로 풀어보시면 좋습니다.
